Лабораторная работа №6

Соответствует лекции: Лекция №4 — Программный интерфейс (API) моделей Claude

🎯 30 шагов выполнения

1
Откройте терминал (Command Prompt на Windows или Terminal на macOS/Linux).
2
Убедитесь, что Python версии 3.9 или выше установлен: выполните команду `python --version`.
3
Если Python не установлен, скачайте его с официального сайта python.org и установите.
4
В терминале выполните команду: `pip install anthropic`.
5
Дождитесь завершения установки (появится сообщение «Successfully installed»).
6
Создайте папку на рабочем столе с именем `claude_lab6`.
7
Откройте текстовый редактор (VS Code, PyCharm или Блокнот).
8
Создайте файл `config.py` и вставьте код: `ANTHROPIC_API_KEY = "sk-ant-..."`
9
Замените `"sk-ant-..."` на ваш реальный API-ключ (получите на console.anthropic.com).
10
Сохраните файл в папку `claude_lab6`.
11
Создайте файл `first_request.py` с кодом из примера ниже.
12
import anthropic
from config import ANTHROPIC_API_KEY

client = anthropic.Anthropic(api_key=ANTHROPIC_API_KEY)

response = client.messages.create(
model="claude-3-haiku-20240307",
max_tokens=100,
temperature=0,
messages=[{"role": "user", "content": "Назовите три столицы европейских государств"}]
)

print(response.content[0].text)
print(f"Входные токены: {response.usage.input_tokens}")
print(f"Выходные токены: {response.usage.output_tokens}")
13
Сохраните файл в папку `claude_lab6`.
14

15

16
Зафиксируйте полученный ответ в файл `lab6_response.txt`.
17

18
Создайте файл `temperature_experiment.py` с кодом из примера ниже.
19
import anthropic
from config import ANTHROPIC_API_KEY

client = anthropic.Anthropic(api_key=ANTHROPIC_API_KEY)

PROMPT = "Придумайте короткий креативный заголовок для статьи об ИИ"
TEMPERATURES = [0.0, 0.5, 1.0]

for temp in TEMPERATURES:
print(f"\nТемпература = {temp}")
for i in range(3):
response = client.messages.create(
model="claude-3-haiku-20240307",
max_tokens=50,
temperature=temp,
messages=[{"role": "user", "content": PROMPT}]
)
print(f" {i+1}: {response.content[0].text}")
